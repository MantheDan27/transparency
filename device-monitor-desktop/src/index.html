Privacy & Data

            
Complete transparency about what Transparency stores, transmits, and how to manage your data.

          

        


        
        

          
Zero Telemetry Architecture

          
Transparency collects no usage analytics, sends no data to external servers, and requires no account or registration to use. All data — device history, alert rules, scan results, notes — is stored locally on this device and never transmitted off your network.

          

            
            Last external network call made: never
          

          
What is stored locally

          
    
            
  • Device discovery results (IP, MAC, hostname, vendor, ports, services)
    • 
            
  • Custom names, notes, trust levels, tags, and device locations
    • 
            
  • Alert rules and alert history
    • 
            
  • Scan snapshots for comparison
    • 
            
  • Latency history and device presence timeline
    • 
            
  • Monitoring configuration and scheduled scans
    • 
          

          
What is NOT stored

          
    
            
  • No browsing history, traffic content, or packet payloads
    • 
            
  • No cloud accounts, authentication tokens, or personal information
    • 
            
  • No analytics, telemetry, or usage statistics
    • 
          

          
No Ads, Ever

          
No banner ads, sponsored suggestions, or affiliate placements. The business model is a one-time purchase — not a subscription, not ad-supported.

Local REST API

          

          

            

              A local-only REST API (no cloud, no external calls) for integration with Home Assistant, Grafana, Node-RED, and custom scripts.
            

            

              http://localhost:7722
              
            

            

              API Key

Example API Calls

              
# List all devices
curl -H "X-API-Key: YOUR_KEY" http://localhost:7722/api/devices

# Get alerts
curl -H "X-API-Key: YOUR_KEY" http://localhost:7722/api/alerts

# Trigger a scan
curl -X POST -H "X-API-Key: YOUR_KEY" http://localhost:7722/api/scan

# Get scan snapshots
curl -H "X-API-Key: YOUR_KEY" http://localhost:7722/api/snapshots

# Get network info
curl -H "X-API-Key: YOUR_KEY" http://localhost:7722/api/network
