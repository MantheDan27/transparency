How Transparency Works

Transparency scans your local network only. All scan data stays on your machine unless you explicitly use the Enrich feature on a device.

What Enrichment sends to the cloud

  • network_metadata — Device IP address on your local network
  • device_identity — Device hostname (if DNS-resolvable)
  • service_inventory — List of open TCP ports detected by this scanner

The cloud service returns defensive guidance only — no exploit instructions, no external scanning, no personal information stored beyond the session.

Cloud enrichment mode

When Cloud Enrichment is enabled, a data preview is shown before any information is transmitted. You can review exactly what will be sent and choose to cancel.

Your rights

You may delete any or all cloud records at any time. Every deletion is permanently logged in the Data Ledger for full transparency. Use the Data Ledger tab for per-device controls.
