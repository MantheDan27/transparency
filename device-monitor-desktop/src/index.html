How Transparency Works

Transparency scans your local network only. All scan data stays on your machine unless you explicitly use the Enrich feature on a device.

What Enrichment sends to the cloud

  • Device IP address
  • Device hostname (if DNS-resolvable)
  • List of open network ports detected by this scanner

The cloud service returns defensive guidance only — no exploit instructions, no external scanning, no personal information stored beyond the session.

Your rights

You may delete any or all cloud records at any time using the controls below. Every deletion is permanently logged in the Data Ledger for full transparency.

Data Deletion

Delete Data for a Specific Device

Remove cloud enrichment records for a single device IP.

Delete All Cloud Data

Permanently removes all enrichment history. This action is irreversible and will be logged.

Clear Local Scan Snapshot

Resets the baseline used to detect new devices and port changes. The next scan will start fresh without comparing to previous results.
